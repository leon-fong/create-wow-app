## Getting Started
To scaffold an app using `create-wow-app`, run any of the following four commands and answer the command prompt questions:

With NPM:

```bash
$ npm create wow-app@latest
```

With Yarn:

```bash
$ yarn create wow-app
```

With PNPM:

```bash
$ pnpm create wow-app
```

With Bun:

```bash
$ bun create wow-app
```
